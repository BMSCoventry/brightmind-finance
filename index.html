First-time setup
Your data will be saved as a JSON file in your GitHub repo — no third-party servers, all under your control.

  1. Go to github.com/settings/tokens/new
  2. Select Classic token, give it any name
  3. Tick the repo checkbox (full repo access)
  4. Click Generate token and paste it below
Choose a password. Anyone visiting your GitHub Pages URL will need to enter this to see the app.
